Основной функционал:

2. Увидеть информацию о том, что заем успешно оформлен - уведомление тостом
3. Видеть историю оформленных займов с актуальным статусом займа - все займы с их статусом, суммой и датой оформления можно увидеть списком на соотвествующем экране
4. Видеть детальную иформацию о каждом оформленном займе и инструкцию как его получить в банке/оформить на карту - сделано
5. Видеть необходимую информацию о произошедших ошибках, не нужно показывать юзеру текст из Exception - юзеру будет показан либо тост, либо снэкбар с переведенной ошибкой
